# iOS ショートカット設定ガイド

iPhoneの「ショートカット」アプリを使って、着信番号をすばやく検索できるようにします。

> **事前準備**: GitHub Pages にデプロイ済みであること。
> 以下の `<YOUR_GITHUB_PAGES_URL>` を実際のURLに置き換えてください。
> 例: `https://username.github.io/phone-search`

---

## パターンA: クリップボードから検索（推奨）

着信履歴から番号をコピー → ショートカット実行 → 結果表示

### 設定手順

1. **ショートカット**アプリを開く
2. 右上の **+** をタップ → 新規ショートカット作成
3. 以下のアクションを順番に追加:

```
┌─────────────────────────────────────┐
│  ショートカット名: 電話番号検索      │
├─────────────────────────────────────┤
│                                     │
│  1. [クリップボードを取得]           │
│     ↓                               │
│  2. [URLを開く]                     │
│     URL: <YOUR_GITHUB_PAGES_URL>    │
│          /?number=[クリップボード]    │
│                                     │
└─────────────────────────────────────┘
```

### 詳細手順

1. **「アクションを追加」** をタップ
2. 検索バーに **「クリップボード」** と入力
3. **「クリップボードを取得」** を選択
4. **「+」** で次のアクションを追加
5. 検索バーに **「URL」** と入力
6. **「URLを開く」** を選択
7. URLフィールドに以下を入力:
   ```
   <YOUR_GITHUB_PAGES_URL>/?number=
   ```
8. URLの末尾にカーソルを置き、上の変数バーから **「クリップボード」** をタップして挿入
9. ショートカット名を **「電話番号検索」** に変更
10. **完了** をタップ

### 使い方

```
┌──────────┐    ┌──────────┐    ┌──────────┐
│  電話    │    │ ショート  │    │  検索    │
│  着信履歴│ →  │ カット    │ →  │  結果    │
│  番号コピー│   │  実行    │    │  表示    │
└──────────┘    └──────────┘    └──────────┘
```

1. **電話**アプリ → **履歴** → 番号を**長押し** → **コピー**
2. **ショートカット**を実行（ウィジェット or ホーム画面アイコン）
3. 自動的にブラウザが開き、検索結果が表示される

## パターンB: 手入力で検索

番号を手入力して検索する方法です。

### 設定手順

```
┌─────────────────────────────────────┐
│  ショートカット名: 電話番号を調べる   │
├─────────────────────────────────────┤
│                                     │
│  1. [入力を要求]                    │
│     プロンプト:                      │
│     "電話番号を入力してください"      │
│     入力タイプ: テキスト             │
│     ↓                               │
│  2. [URLを開く]                     │
│     URL: <YOUR_GITHUB_PAGES_URL>    │
│          /?number=[入力結果]         │
│                                     │
└─────────────────────────────────────┘
```

### 詳細手順

1. **「アクションを追加」** をタップ
2. 検索バーに **「入力」** と入力
3. **「入力を要求」** を選択
4. プロンプトに **「電話番号を入力してください」** と入力
5. **「+」** で次のアクションを追加
6. **「URLを開く」** を選択
7. URLフィールドに以下を入力:
   ```
   <YOUR_GITHUB_PAGES_URL>/?number=
   ```
8. URLの末尾にカーソルを置き、変数バーから **「入力を要求の結果」** をタップして挿入
9. ショートカット名を **「電話番号を調べる」** に変更
10. **完了** をタップ

## ホーム画面に追加

どちらのパターンも、ホーム画面に追加するとワンタップで起動できます。

1. ショートカット一覧で該当のショートカットを**長押し**
2. **「詳細」** をタップ
3. **「ホーム画面に追加」** をタップ
4. アイコン・名前を設定して **「追加」**

### ウィジェットとして追加

1. ホーム画面を**長押し** → 左上の **+** をタップ
2. **「ショートカット」** ウィジェットを検索
3. サイズを選択して **「ウィジェットを追加」**
4. ウィジェットをタップして表示するショートカットを選択
